Prerequisites:
- Visual Studio 2015 Update 3. 
- MS SQL server (express) 2008R2+.
- .Net Framework 4.6.1.
- SQL Server managment studio 2008R2+.
- IISExpress 10.

Before run solution:
1. In projects NCD_Web/NCD_WebService in web.configs you need to write correct sqlConnectionString (Database name: NationalCriminalsDatabase).
2. With the help of managment studio run sql script NCD_Model/TestData.sql (Database scheme with test data).
3. Update Nuget packeges if needed.
